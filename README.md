# Evaluación Servidores

Este repositorio contiene la página web desarrollada en el servidor Linux como parte de la evaluación.

## Contenido

- index.html: Página web alojada en el servidor Apache (httpd)

## Descripción

Se configuró un servidor web en Linux utilizando Apache, donde se creó una página HTML simple para verificar el funcionamiento del servicio.

La página fue probada:
- Localmente en Linux (http://localhost)
- Desde una máquina Windows mediante IP

## Resultado

El servidor web funciona correctamente y permite acceso desde otras máquinas en la red.
